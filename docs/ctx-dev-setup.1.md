# NAME

ctx-dev-setup - Set up CTX OS developer environment

# SYNOPSIS

`ctx-dev-setup`

# DESCRIPTION

`ctx-dev-setup` installs common development tools on CTX OS, including:
- Build essentials (build-essential, gcc, make)
- Version control (git)
- Common utilities (curl, wget, jq, tree)
- Editors (vim)
- System tools (htop)

The script uses sudo to install packages and handles errors gracefully.
